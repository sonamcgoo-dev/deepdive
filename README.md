# Deep Dive Web Scraper

A skill for OpenHands that performs comprehensive web scraping and research.

## Quick Start

### Installation

In any OpenHands conversation, type:
```
/add-skill https://github.com/sonamcgoo-dev/deepdive/tree/main/skills/deepdive
```

Or manually copy `SKILL.md` to your workspace's `.agents/skills/deepdive/` folder.

### Usage

1. Start any OpenHands conversation
2. Type **`deepdive`** followed by a topic or URL
3. The skill will scrape, research, and generate a report
4. A markdown report is saved to your workspace

## Examples

**Scrape a website:**
> "deepdive https://example.com"

**Research a topic:**
> "deepdive AI developments in healthcare"

**Deep analysis:**
> "deepdive everything about cryptocurrency regulations"

## Output

Generates a markdown report with:
- Executive summary
- Key findings
- Detailed analysis
- Extracted data
- Source list
- Conclusion

## Dark Web

If you have Tor running, you can access `.onion` sites:
> "deepdive tor://example.onion/page"

Note: Requires Tor daemon on localhost:9050
